# OPSKE Automation Agent 🤖

Ένα εργαλείο αυτοματισμού (bot) γραμμένο σε Python για την μαζική υποβολή δικαιολογητικών στην πλατφόρμα του ΟΠΣΚΕ (opske.gr).

Η εφαρμογή έχει σχεδιαστεί για να εξοικονομεί χρόνο αυτοματοποιώντας επαναλαμβανόμενες διαδικασίες εισαγωγής δεδομένων και μεταφόρτωσης αρχείων, διασφαλίζοντας παράλληλα την ορθότητα των στοιχείων μέσω ελέγχων εγκυρότητας.

## ✨ Δυνατότητες

* **Ασφαλής Σύνδεση:** Login μέσω modal παραθύρου (τα credentials δεν αποθηκεύονται στον κώδικα).
* **Ανάγνωση Δεδομένων:** Μαζική εισαγωγή στοιχείων από αρχεία Excel.
* **Έλεγχος Εγκυρότητας (Validation):**
    * Έλεγχος εγκυρότητας ΑΦΜ (αλγόριθμος ελέγχου ψηφίου).
    * Έλεγχος τύπου και μεγέθους αρχείων.
    * Έλεγχος ημερομηνιών.
* **GUI:** Φιλικό γραφικό περιβάλλον χρήστη (Tkinter) για εύκολη χρήση.
* **Αυτοματισμός Browser:** Χρήση της βιβλιοθήκης Playwright για γρήγορη και αξιόπιστη αλληλεπίδραση με την ιστοσελίδα.

## 🛠️ Τεχνολογίες

* Python 3
* Playwright (Browser Automation)
* Tkinter (GUI)
* Pandas (Data Manipulation)
* OpenPyXL (Excel Support)

## 🚀 Εγκατάσταση & Χρήση

1.  **Εγκατάσταση βιβλιοθηκών:**
    ```bash
    pip install -r requirements.txt
    playwright install chromium
    ```

2.  **Εκτέλεση:**
    ```bash
    opske_automatation_agent.py
    ```

3.  **Λειτουργία:**
    * Εισάγετε τα στοιχεία σύνδεσης στο popup παράθυρο.
    * Επιλέξτε το αρχείο Excel με τα δεδομένα.
    * Επιλέξτε τον φάκελο που περιέχει τα σαρωμένα δικαιολογητικά.
    * Πατήστε "Έναρξη".
